🎓 Udemy Clone Front-End Website
A responsive Udemy-inspired front-end project built using HTML and CSS, designed to replicate the look and feel of a modern online learning platform.

💡 Key Features
Clean and intuitive user interface with category navigation

Hero banner highlighting essential career and life skills

Responsive course recommendation section with dynamic layout

Consistent color palette and typography for a professional finish

🧠 Objective
This project demonstrates proficiency in front-end development, focusing on semantic HTML, CSS Flexbox/Grid, and UI/UX design principles. It serves as a showcase of layout structuring and styling for educational platforms.

⚙️ Technologies Used
HTML5

CSS3
