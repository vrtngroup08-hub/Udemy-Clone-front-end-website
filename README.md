🎓 Udemy Clone Front-End Website
A responsive Udemy-inspired front-end website built using HTML and CSS, designed to replicate the look and feel of a modern online learning platform. This project focuses on clean design, usability, and responsive layout techniques.

✨ Features
Interactive navigation bar with categorized buttons

Hero section with gradient background and promotional banner

“Recommended for You” course grid with responsive cards

Consistent color palette and typography for a professional finish

🧠 Purpose
This project demonstrates front-end development skills in semantic HTML, CSS Flexbox/Grid, and UI/UX design principles. It serves as a showcase of layout structuring and styling for educational platforms.

⚙️ Technologies Used
HTML5

CSS3
